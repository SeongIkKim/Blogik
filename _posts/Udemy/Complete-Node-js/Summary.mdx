# Node.js

## 구성
npm : Node Packaged Manager, Node.js용 패키지 관리 툴
npm_module : npm에 설치된 (로컬)패키지가 저장되는 곳. -g 옵션으로 설치 시 글로벌 패키지 폴더에 저장
package.json : node.js의 의존성 패키지 관리 문서

## Packages

* `nodemon` : node 파일 변경사항을 자동 적용/재실행
